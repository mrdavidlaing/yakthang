Switched from mtime-based detection to git notes hash detection. The timer now reads /host/.git/refs/notes/yaks (~40 bytes) on each tick and compares the commit hash to the cached value. Since every yx operation commits to refs/notes/yaks, this catches ALL changes — field updates, state transitions, nested adds, removals. The 'r' key still forces a full rebuild. Build (wasm32-wasip1) and all 70 tests pass.
