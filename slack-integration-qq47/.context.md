# Phase 6: Slack Integration (1-2 hours)

Create a Slack app and configure OpenClaw to receive messages via Socket Mode.

## Steps

1. Go to api.slack.com/apps → Create New App
2. Enable Socket Mode
3. Add Event Subscriptions: message.im, app_mention
4. Install to workspace
5. Copy App-Level Token (xapp-...) and Bot Token (xoxb-...)

6. Add to ~/.openclaw/openclaw.json:
```json5
{
  channels: {
    slack: {
      enabled: true,
      mode: "socket",
      appToken: "xapp-...",
      botToken: "xoxb-...",
      dm: {
        enabled: true,
        policy: "allowlist",
        allowFrom: ["U_YOUR_USER_ID"]
      }
    }
  }
}
```

7. Restart Gateway: systemctl restart openclaw-gateway

## Done When
- Slack app created and installed in workspace
- Tokens configured in openclaw.json
- DM to Yakob bot gets a response
