Background: Claude Code auto mode (--permission-mode auto) uses a background
AI classifier (Sonnet 4.6) to evaluate each action against block/allow rules.
It blocks data exfiltration, destructive git ops, credential misuse — things
that filesystem/network sandboxing can't judge. Combined with srt, this gives
defense in depth.

See: https://www.anthropic.com/engineering/claude-code-auto-mode
