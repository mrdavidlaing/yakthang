## Investigate ast-grep (https://ast-grep.github.io/)

ast-grep is a CLI tool for structural search/replace using AST patterns. 
Like grep but understands code structure, not just text.

### Questions to answer
- What would ast-grep add over ripgrep/standard grep for yakthang workflows?
- Could it be useful for shavers doing code refactoring tasks?
- Could it power smarter code analysis in skills or review agents?
- How would it be installed/distributed (nix, npm, binary)?
