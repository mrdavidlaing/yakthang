## What
Once mattwynne merges PR https://github.com/mattwynne/yaks/pull/4, update the yaks submodule in yakthang to pick up the new done checkmark icon.

## Steps
1. Check if PR #4 is merged: gh pr view 4 --repo mattwynne/yaks --json state
2. If merged, update submodule: cd src/yaks && git fetch origin && git checkout main && git pull
3. cd back to repo root, commit the submodule pointer update
4. Verify yx ls now shows ✓ for done yaks
