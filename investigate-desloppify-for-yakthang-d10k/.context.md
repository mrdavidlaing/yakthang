Investigate desloppify (https://github.com/peteromallet/desloppify) on the yak-map Rust codebase.

desloppify is an agent harness that systematically finds and fixes code quality issues — both mechanical (dead code, duplication, complexity) and subjective (naming, abstractions, module boundaries). Uses LLM evaluation + traditional linters.

## Tasks
1. Install desloppify (pip install desloppify, requires Python 3.11+)
2. Run 'desloppify scan --path .' from within src/yak-map
3. Review the scan results — what does it find? What's the score?
4. Rust has generic linter + tree-sitter support (not full plugin depth). Note any coverage gaps.
5. Try running 'desloppify next' to see what it recommends fixing first.

## Deliverable
Write findings as a note on this yak via:
  yx field investigate desloppify for yakthang notes="<summary>"
Include: scan results, score, language coverage assessment for Rust, and recommendation (adopt / skip / revisit later).
