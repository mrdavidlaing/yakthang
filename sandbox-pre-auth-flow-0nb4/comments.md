Added copyHostOAuthCredentials() to spawn_sandbox.go. It runs after setupClaudeSettings() and copies .json files from the host's ~/.claude/ into the worker's .claude/ dir, skipping managed files (.claude.json, settings.json, remote-settings.json, api-key-helper.sh). Files already present in the worker dir are not overwritten. This gives sandbox workers working OAuth tokens so they don't need to complete a localhost redirect flow that bubblewrap's network namespace blocks.
