# Update Stop Logic

Modify `src/yak-box/cmd/stop.go` to unregister sessions instead of deleting .meta files.

## Changes

Replace:
```go
if err := metadata.DeleteMetadata(stopName); err != nil {
    fmt.Printf("Warning: Failed to delete metadata: %v\n", err)
}
```

With:
```go
if err := sessions.Unregister(yakBoxesDir, stopName); err != nil {
    fmt.Printf("Warning: Failed to unregister session: %v\n", err)
}
```

## Keep Home Directory Intact

Do NOT delete `.yak-boxes/@home/{persona}/` on stop. This is the whole point - persistent home across sessions.

## Remove Old Metadata Code

Remove imports and references to the old `metadata` package since we're moving to `sessions`.

## Testing

- Stop a worker
- Verify `sessions.json` entry removed
- Verify `.yak-boxes/@home/{persona}/` still exists
- Verify subsequent spawn reuses the same home directory
