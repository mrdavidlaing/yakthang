## Pattern Being Learned
**Compiler as quality tool** — yx lets the Rust compiler surface all warnings. Suppressing them hides dead code and unused imports.

## Process

2. Run cargo check (both native and WASM targets)
3. Fix any warnings that surface — each one is a small cleanup
4. Run bin/dev check (if it exists) to verify

- All existing tests pass
