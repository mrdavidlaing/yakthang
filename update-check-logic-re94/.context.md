# Update Check Logic

Modify `src/yak-box/cmd/check.go` to read from `sessions.json`.

## Changes

Replace metadata loading:
```go
meta, err := metadata.LoadMetadata(workerName)
```

With sessions loading:
```go
session, err := sessions.Get(yakBoxesDir, workerName)
```

Update output to use session data:
```go
fmt.Printf("Worker: %s (%s)\n", session.Worker, session.Name)
fmt.Printf("Task: %s\n", session.Task)
fmt.Printf("Container: %s\n", session.Container)
// ... etc
```

## List All Workers

For `check --wip` and `check --blocked`:
```go
activeSessions, err := sessions.List(yakBoxesDir)
```

## Testing

- Run `./bin/yak-box check` with active workers
- Verify output shows correct worker/task info
- Run `./bin/yak-box check --wip`
- Run `./bin/yak-box check --blocked`
