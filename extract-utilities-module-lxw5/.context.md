Move standalone utility functions to src/yak-map/src/util.rs.

## Pattern Being Learned
**Separate infrastructure from domain** — yx puts cross-cutting utilities in infrastructure/. For yak-map, a util.rs module is the first step toward multi-file structure.

## What to Move
Functions with zero dependencies on State or TaskRepository:
- escape_single_quoted() (lines 169-182)
- copy_via_zellij_tty() (lines 187-204)
- base64_encode() (lines 206-220)
- strip_ansi() (lines 229-246)
- line_display_width() (lines 224-226)

## Process
1. Create src/yak-map/src/util.rs
2. Move functions + their tests
3. Add mod util; to main.rs
4. Update imports — use util::{...}
5. Verify cargo check + cargo test

## Definition of Done
- util.rs exists with the 5 functions
- main.rs uses them via use util::...
- All tests pass (including any tests that moved)
- main.rs is ~100 lines shorter
