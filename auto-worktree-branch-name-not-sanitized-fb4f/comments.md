Added SanitizeBranchName to pkg/worktree/manager.go — lowercases, replaces spaces and invalid git branch chars with hyphens, collapses consecutive hyphens, trims leading/trailing hyphens and dots. Applied it in EnsureWorktree (the --auto-worktree path) and in resolveInheritedWorktrees (the inherited worktrees path). Both paths now produce valid git branch names from arbitrary yak names like 'SC-102103 add UnixODBC manager to imogen-workstation'. Added 11 test cases covering spaces, uppercase, special chars, consecutive hyphens, edge cases.
