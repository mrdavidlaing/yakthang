# Create Sessions Package

Create `src/yak-box/internal/sessions/` package to manage `sessions.json`.

## Package Interface

```go
package sessions

import "time"

type Session struct {
    Name      string    `json:"name"`        // e.g., "compile-yx"
    Worker    string    `json:"worker"`      // e.g., "yakriel"
    Task      string    `json:"task"`        // e.g., "yakthang-v2/compile-yx"
    Container string    `json:"container"`   // e.g., "yak-shaver-compile-yx"
    SpawnedAt time.Time `json:"spawned_at"`
    Runtime   string    `json:"runtime"`     // "sandboxed" or "native"
    CWD       string    `json:"cwd"`
    // Add any other fields from current metadata
}

// Register adds or updates a session
func Register(yakBoxesDir string, session Session) error

// Unregister removes a session
func Unregister(yakBoxesDir string, name string) error

// Get retrieves a specific session
func Get(yakBoxesDir string, name string) (*Session, error)

// List returns all active sessions
func List(yakBoxesDir string) ([]Session, error)
```

## File Location

- File: `.yak-boxes/sessions.json`
- Use file locking to prevent concurrent access corruption
- Create atomically (write to temp, rename)

## Error Handling

- Return clear errors if file is missing/corrupt
- Initialize empty sessions.json if doesn't exist
