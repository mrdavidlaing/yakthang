Build yak-map: a Zellij plugin for visualizing the task tree.

yak-map replaces the current yak-map.sh script with a proper Zellij WASM plugin (Rust).

Reference:
- docs/orchestrator-layout.md for how the orchestrator UI is structured
- The yx CLI (task tracker) for understanding the task tree format
- Zellij plugin SDK docs: https://zellij.dev/documentation/plugins

Phase 1 - Scaffold:
1. Initialize a Rust project in src/yak-map/ (cargo init --lib)
2. Add zellij-tile dependency
3. Create basic plugin that renders a placeholder in Zellij
4. Create a design doc at docs/yak-map-design.md outlining planned features
5. Read existing docs to understand what yak-map should display

Start with the scaffold and design doc. Do NOT implement full functionality yet.
