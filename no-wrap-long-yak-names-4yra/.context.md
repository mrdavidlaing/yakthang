In the yak-map Zellij plugin, long yak names currently wrap to the next line when the pane is narrow. Instead, they should truncate/clip at the pane edge — the text just disappears off the right side. The user can widen the Zellij pane to see the full name if needed.

## Acceptance criteria
- Long yak names do not wrap to a second line
- Text is clipped at the pane boundary (no ellipsis needed, just cut off)
- Widening the pane reveals more of the name
- Existing tests pass
