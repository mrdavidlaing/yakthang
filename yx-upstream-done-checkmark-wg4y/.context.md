## What
Submit a PR to mattwynne/yaks to change the done icon from ● to ✓ in yx ls output.

## Approach
1. Clone mrdavidlaing/yaks fork (or create fork if needed)
2. Add mattwynne/yaks as upstream remote, sync to latest
3. Create a feature branch
4. Change done icon from ● to ✓ in:
   - src/adapters/user_display/mod.rs (~line 62-75, style_yak_item function)
   - src/adapters/tui_display/mod.rs (~line 126-131, indicator_for function)
5. Update any tests that assert on the done icon character
6. Commit, push to mrdavidlaing/yaks fork
7. Create PR from mrdavidlaing/yaks to mattwynne/yaks

## Key detail
The submodule in yakthang points to mattwynne/yaks.git. We do NOT change the submodule yet — we wait for the upstream PR to be merged, then update.

## Acceptance criteria
- PR created against mattwynne/yaks
- Done yaks show ✓ in yx ls output
- WIP still shows ●, todo still shows ○
- All tests pass
