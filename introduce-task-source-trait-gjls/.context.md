Introduce a TaskSource trait to enable in-memory testing.

## Pattern Being Learned
**Ports and adapters (hexagonal architecture)** — This is the core yx pattern. yx defines ReadYakStore and WriteYakStore traits (src/yaks/src/domain/ports/yak_store.rs) with InMemoryStorage for testing and DirectoryStorage for production. Contract tests verify both behave identically.

**Preparing-a-yak** — Spec this with the preparing-a-yak skill. This is an architectural decision — write an ADR.

## What to Change
TaskRepository (lines 15-107) is a concrete struct reading the filesystem. Tests use tempfile::TempDir.

Extract a trait:
  trait TaskSource {
      fn list_tasks(&self) -> Vec<TaskInfo>;
      fn get_field(&self, task: &str, field: &str) -> Option<String>;
  }

Create InMemoryTaskSource for tests. State takes Box<dyn TaskSource>.
