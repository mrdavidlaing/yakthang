## Goal

Clean up `src/yak-map` and `src/yak-box` so they feel like monorepo packages, not standalone repos. `src/yaks` stays as a submodule (upstream mattwynne/yaks).

## Current state

Both `src/yak-map` and `src/yak-box` are already tracked by the parent repo's git ‚Äî no submodule extraction needed. But they still carry artifacts from when they were (or were designed as) standalone repos.

## Changes needed

### src/yak-box

1. **Delete `src/yak-box/.claude/`** ‚Äî has its own `settings.local.json` and `skills/desloppify/`. Standalone repo artifact. If desloppify is useful, move it to top-level `skills/` first.
2. **Rewrite `src/yak-box/AGENTS.md`** ‚Äî should be the canonical doc. Concise monorepo-appropriate build & test notes (Go build, shellspec tests, devcontainer purpose). Remove OpenCode/rtk/showboat references.
3. **`src/yak-box/CLAUDE.md`** ‚Äî replace with a symlink to `AGENTS.md` (`ln -s AGENTS.md CLAUDE.md`).
4. **Consolidate `src/yak-box/.gitignore`** ‚Äî move the `yak-box` binary ignore to top-level `.gitignore` (under `src/yak-box/yak-box`). Keep `.yaks` and `.desloppify` ignores only if still relevant.
5. **Keep `src/yak-box/.devcontainer/`** ‚Äî this is used at runtime by yak-box to build sandboxed worker images. It's intentional, not a standalone repo artifact.
6. **Keep `src/yak-box/.shellspec`** ‚Äî test config is fine, just verify tests run from monorepo root: `cd src/yak-box && shellspec`
7. **Update Go module path** ‚Äî change `github.com/wellmaintained/yak-box` to `github.com/wellmaintained/yakthang/src/yak-box` in `go.mod` and all internal imports.

### src/yak-map

1. **`src/yak-map/AGENTS.md`** ‚Äî keep as the canonical doc. Trim to concise monorepo-appropriate build/test instructions.
2. **`src/yak-map/CLAUDE.md`** ‚Äî replace with a symlink to `AGENTS.md` (`ln -s AGENTS.md CLAUDE.md`).

### Top-level

1. **Update top-level `.gitignore`** ‚Äî add `src/yak-box/yak-box` (build output) if not already there.
2. **Verify Justfile still works** ‚Äî `just build` and `just clean` should succeed after changes.

## Out of scope
