for sandbox workers, which means it can't:
- Kill the process tree (no PID file found)
- Clean up the srt config temp file
- Close the Zellij tab

Error: "Warning: failed to find worker home dir: worker home dir not found for <name>"

The worker home dir lives at .yak-boxes/@home/<shaver-name>/. StopSandboxWorker
uses findWorkerHomeDir() which walks up from the CWD looking for a .yak-boxes
directory. This may be failing because:

1. The CWD stored in sessions.json doesn't have .yak-boxes/ above it
2. The function is looking for the yak-name not the shaver-name
3. The function's walk-up logic doesn't match the actual directory structure
