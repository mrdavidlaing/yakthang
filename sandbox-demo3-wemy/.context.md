Smoke test: spawn a sandboxed (Docker) worker without an API key to test the OAuth login flow inside a container.

## What to do
1. You should have been prompted to complete OAuth login via device flow
2. Once authenticated, run: echo 'Hello from a sandboxed shaver\!' 
3. Run: yx ls (verify you can see the yak map)
4. Mark yourself done: yx done sandbox-demo3

## Acceptance criteria
- Sandboxed container starts without _ANTHROPIC_API_KEY
- OAuth device flow completes successfully
- Claude Code is functional inside the container
