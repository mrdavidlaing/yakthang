Added @anthropic-ai/sandbox-runtime as git submodule at src/sandbox-runtime. Added build-srt and install-srt Justfile targets; both wired into aggregate build/install targets. install-srt handles Linux seccomp vendor binaries (x64/arm64) by copying them alongside the binary at ~/.local/bin/vendor/seccomp/. macOS needs no vendor files. Updated .devcontainer/Dockerfile to install just and bun; added postCreateCommand to init submodules and build srt. Note: used `bun install` without --frozen-lockfile since upstream uses package-lock.json (npm format) and bun migrates it on first run.
