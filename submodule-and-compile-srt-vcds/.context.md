Add @anthropic-ai/sandbox-runtime as a git submodule and compile with Bun.

Steps:
1. git submodule add https://github.com/anthropic-experimental/sandbox-runtime src/sandbox-runtime
2. Add Justfile targets:
   - `build-srt`: cd src/sandbox-runtime && bun build --compile src/cli.ts --outfile srt
   - `install-srt`: cp src/sandbox-runtime/srt ~/.local/bin/srt
3. Add to `just build` and `just install` aggregate targets
4. Document Linux seccomp vendor binaries caveat:
   - bun compile bundles code but not vendor/ assets
   - For Linux: copy vendor/seccomp/ alongside binary, or set explicit paths in srt config
   - macOS: no vendor files needed (uses sandbox-exec)
5. Update .devcontainer/Dockerfile to build srt during image creation

Relevant files:
- .gitmodules (add entry)
- Justfile (add targets)
- .devcontainer/Dockerfile (add build step)

Definition of Done:
- src/sandbox-runtime/ exists as submodule
- `just build-srt` produces a standalone srt binary
- `just install-srt` copies to ~/.local/bin/
- srt --help runs successfully from ~/.local/bin/srt
- Seccomp vendor files handled for Linux
