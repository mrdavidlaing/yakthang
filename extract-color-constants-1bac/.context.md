Replace scattered ANSI escape codes with named constants.

## Pattern Being Learned
**Named constants over magic values** — yx uses structured display via DisplayPort trait. For yak-map, start simpler: just name the constants.

## What to Change
Lines 340-453 of main.rs have raw ANSI codes like "\x1b[31m", "\x1b[0m", background color numbers (237, etc.) scattered across task_color(), render_task(), tree_prefix(), highlight_line().

## Process
1. Inventory all ANSI codes used (grep for \x1b)
2. Create a const block or module with named constants (RED, RESET, BG_SELECTED, etc.)
3. Replace all raw codes with the named constants
4. Run tests — rendering output should be byte-identical

## Definition of Done
- Zero raw ANSI escape codes outside the constants block
- All tests pass
- Rendering unchanged (visual diff)
