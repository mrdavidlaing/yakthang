# Update Spawn Logic

Modify `src/yak-box/internal/runtime/sandboxed.go` and `src/yak-box/cmd/spawn.go`.

## Changes to sandboxed.go

```go
func SpawnSandboxedWorker(worker *types.Worker, persona *types.Persona, prompt string) error {
    // 1. Create @home/{persona}/ directory
    workerHome := filepath.Join(workspaceRoot, ".yak-boxes", "@home", persona.Name)
    if err := os.MkdirAll(workerHome, 0755); err != nil {
        return fmt.Errorf("failed to create worker home: %w", err)
    }
    
    // 2. Mount worker's home directory
    dockerArgs = append(dockerArgs,
        "-v", fmt.Sprintf("%s:/home/yakshaver", workerHome),
    )
    
    // 3. Set YAK_PATH environment variable
    dockerArgs = append(dockerArgs,
        "-e", "YAK_PATH=/workspace/.yaks",
    )
    
    // ... rest of spawn logic ...
}
```

## Changes to spawn.go

After successful container start:

```go
// Register session in sessions.json
session := sessions.Session{
    Name:      spawnName,
    Worker:    persona.Name,
    Task:      spawnTask,
    Container: containerName,
    SpawnedAt: time.Now(),
    Runtime:   "sandboxed",
    CWD:       spawnCwd,
}
if err := sessions.Register(yakBoxesDir, session); err != nil {
    // Log warning but don't fail spawn
    fmt.Printf("Warning: failed to register session: %v\n", err)
}
```
