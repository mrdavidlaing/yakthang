Track and report LLM API costs across OpenCode workers and OpenClaw Gateway.

Goals:
- Real-time cost visibility for orchestrator and workers
- Per-task cost attribution
- Daily/weekly/monthly cost summaries
- Budget alerts and spending limits

Components:
1. OpenCode worker cost tracking (via .opencode usage tracking)
2. OpenClaw Gateway cost tracking (via session_status and usage APIs)
3. Unified cost reporting dashboard/script
